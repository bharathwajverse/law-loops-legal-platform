# Security Policy

## Reporting a Vulnerability

Please do not use public issue trackers to report security vulnerabilities.

If you discover a security issue, contact the project maintainer privately and provide enough detail to reproduce the issue.

## Supported Versions

This project is a static web application built with HTML, CSS, and JavaScript. Security issues should be reported if they affect the deployed site or user data.

## Response

The maintainer will review security reports and respond as quickly as possible. If necessary, a fix will be released in a follow-up commit.
